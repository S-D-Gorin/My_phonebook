Консольное приложение - телефонный справочник.

Техническое задание:

Реализовать телефонный справочник со следующими возможностями:

1.                   	Вывод постранично записей из справочника на экран
2.                   	Добавление новой записи в справочник
3.                   	Возможность редактирования записей в справочнике
4.                   	Поиск записей по одной или нескольким характеристикам

Требования к программе:

1.                   	Реализация интерфейса через консоль (без веб- или графического интерфейса)
2.                   	Хранение данных должно быть организовано в виде текстового файла, формат 
			которого придумывает сам программист.
3.                   	В справочнике хранится следующая информация: фамилия, имя, отчество, 
			название организации, телефон рабочий, телефон личный (сотовый)

Плюсом будет:

1.                   	аннотирование функций и переменных
2.                   	документирование функций
3.                   	подробно описанный функционал программы
4.                   	размещение готовой программы и примера файла с данными на github


Программа запускается в консоли. Если в файле bd.txt отсутствуют какие либо записи, 
то программа предлагает создать первую запись, далее запускатся интерфейс взаимодействия с программой
где пользователю предоставляется выбор дальнейших действий:

каждая запись справочника хранится в отдельной строке в следующей последовательности:
1. id или порядковый номер записи
2. Фамилия
3. Имя
4. Отчество
5. Организация
6. Рабочий телефон
7. Личный телефон

Пробел между записями выступает в качестве разделителя и в дальнейшем считывается программой 
для вывода информации пользователю.
